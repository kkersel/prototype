### [2026-06-18] Hover-reveal действий ломается на тач
Ошибка: действия карточки показывались по `:hover` (opacity 0→1) — на тач-устройствах
(нет ховера) они недостижимы.
Решение: прятать/раскрывать по ховеру ТОЛЬКО внутри `@media (hover: hover) and (pointer: fine)`
+ раскрытие по `:focus-within` (клавиатура). Базовое состояние — видимо (opacity по умолчанию).
На тач (iOS/Android → `hover: none`) действия видны всегда.
Правило: любой hover-only UI оборачивать в `@media (hover: hover)`; на тач — видимый аналог.
Заодно: проп варианта компонента не должен утекать в DOM (`variant` → класс, не `{...props}`).

### [2026-06-18] Дизайн-система: уход от «outline + neon dark»
Задача: чистый Figma-like UI до масштабирования.
Решение: токены (tokens.css) + примитивы (components/ui) + светлая тема для рабочих экранов,
тёмная только для плеера. Разделение фоном/тенью/воздухом, иконки вместо эмодзи, segmented
вместо рядов обведённых кнопок.
Правило: новые экраны строить ТОЛЬКО из примитивов и токенов; см. `.ai/design.md`.

### [2026-06-17] ResizeObserver не подхватывается до загрузки данных
Ошибка: тепловая карта схлопывалась в ~50px.
Причина: `useEffect([])` с измерением вешался на mount, когда компонент ещё в раннем
`return <Загрузка…>` и измеряемого узла нет; повторно эффект не запускался.
Решение: callback-ref — наблюдатель цепляется в момент реального монтирования узла.
Правило: измерять DOM через callback-ref, не через `useEffect([])`, если узел появляется условно.

### [2026-06-17] CSS Grid без явных rows «съедает» высоту
Ошибка: в редакторе между топбаром и панелями пустая полоса.
Причина: `display:grid` с `grid-template-columns`, без `grid-template-rows`; лишняя высота
ушла в авто-строку топбара.
Решение: задать `grid-template-rows: 52px 1fr` + `min-height:0` детям.
Правило: для полноэкранных грид-лэйаутов всегда фиксировать rows.
