# Локальные правила проекта

Дополняет global.md. Не противоречит ему.

## Команды
```bash
dev:     npm run dev        # сервер :5174 + Vite :5173 (HMR)
build:   npm run build      # статический билд клиента → dist/
start:   npm start          # прод: сервер отдаёт dist на :5174 (так открывают терминалы)
test:    —                  # авто-тестов пока нет; проверка через preview-инструменты в браузере
lint:    —                  # tsc типы; ESLint не настроен
```

## Тестирование
- Пока ручная проверка в браузере (preview): рендер, консоль без ошибок, прогон флоу.
- Viewport терминалов разный (вкл. квадратный) — проверять адаптивность плеера и стейджа.

## Переменные окружения
```
PORT=5174          # порт сервера (terminals)
HTTPS_PORT=5175    # если есть data/certs/{key,cert}.pem
NODE_ENV=production # включает отдачу dist + SPA-fallback
```

## Специфика проекта
- Терминалы — P10 (Android + Chrome), несколько моделей, разные разрешения, один квадратный.
- Координаты тапов/зон нормализованы 0..1 от холста → тепловые карты разных устройств сводятся.
- Стор файловый (`data/`), бэкап = копия папки.

## Запрещено в этом проекте
- Хардкод цветов/размеров в компонентах — только токены `styles/tokens.css`.
- Эмодзи как иконки — только `components/ui/Icon`.
- `fetch` из страниц в обход `api.ts`.
- Новые зависимости без обсуждения (особенно нативные).
- Рамка (`border`) как способ разделения по умолчанию — фон/тень/воздух.
- Dropdown для контекстных действий на тач (плеер) — только Sheet.
