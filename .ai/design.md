# Дизайн-система — Terminal Prototyper

> Источник истины для UI. Токены — `client/src/styles/tokens.css`, база/утилиты —
> `client/src/styles/global.css`, примитивы — `client/src/components/ui/`.
> UX-состояния — patterns/universal/ui-ux-states.md. Не хардкодить значения в компонентах.

## Философия

Инструмент для дизайнеров → сам должен выглядеть как профессиональный инструмент.
Ориентир — рабочий UI Figma: спокойный, плотный, без украшательства.

1. **Разделяем фоном и воздухом, а не рамками.** Граница (1px) — последнее средство.
   Панели «плавают» на холсте за счёт фона и лёгкой тени, а не обводки. Минимум `border`.
2. **Цвет — это сигнал, а не декор.** Нейтральная серо-белая база; один синий акцент для
   главного действия и выделения; семантика (успех/ошибка) только по делу. Без градиентов
   ради красоты, без неона.
3. **Иконки, а не эмодзи.** Только набор `Icon` (штриховые, 1.5px, currentColor). Эмодзи в UI запрещены.
4. **Иерархия — размером и весом текста + плотностью**, а не разноцветьем.
5. **Каждый экран: loading / empty / error / success.** Действие → мгновенный отклик, без двойного сабмита.
6. **Движение тихое.** 120–200ms, ease-out. Анимации помогают понять переход, не развлекают.

## Тема

Светлая — для рабочих экранов (Главная / Редактор / Тепловые карты).
Тёмная — только для **Плеера**: фуллскрин-устройство на почти-чёрном фоне (корректный letterbox).
Токены сгруппированы так, что тёмная тема для рабочих экранов добавляется через `[data-theme="dark"]`.

## Типографика

Шрифт: **Inter** (системный фолбэк: -apple-system, Segoe UI, Roboto). `font-feature-settings: "cv11","ss01"` для аккуратных цифр/буквы.

| Роль | Токен | Размер / высота | Вес |
|------|-------|-----------------|-----|
| Заголовок страницы | `--font-h1` | 22 / 28 | 680 |
| Заголовок секции | `--font-h2` | 16 / 22 | 620 |
| Подзаголовок панели | `--font-h3` | 13 / 18 | 600 |
| Основной текст | `--font-body` | 14 / 20 | 440 |
| UI-текст (кнопки/строки) | `--font-ui` | 13 / 16 | 500 |
| Метки/капс | `--font-label` | 11.5 / 14, `letter-spacing .04em`, UPPERCASE | 600 |

Заголовки секций-панелей (`--font-label`) — серый, капс, разреженный: «ЭКРАНЫ», «СЕССИИ».

## Пространство, радиусы, тени

- **Шаг**: 4px. Токены `--space-1..-10` = 4,8,12,16,20,24,32,40,48,64.
- **Радиусы**: `--radius-sm 6`, `--radius-md 8`, `--radius-lg 12`, `--radius-xl 16`, `--radius-pill 999`.
- **Тени (элевация, мягкие)**:
  - `--shadow-sm` — карточки/панели на холсте;
  - `--shadow-md` — поповеры, активный элемент segmented;
  - `--shadow-lg` — модалки, шторки.
- **Контейнер**: рабочие экраны — полноширинные грид-лэйауты (десктоп-инструмент).
  Плеер/гейт — центрированная карта.

## Движение
- `--ease` cubic-bezier(.2,.8,.2,1); длительности `--dur-fast 120ms`, `--dur 180ms`, `--dur-slow 240ms`.
- Переходы прототипа (fade/slide/push) — отдельная история плеера, не трогать общие токены.

## Иконки
`components/ui/Icon.tsx` — один набор штриховых SVG (16/20px, `stroke: currentColor`, 1.5).
Имена: plus, play, image, video, layers, target, chevron-{left,right,up,down}, arrow-left,
trash, copy, download, upload, refresh, close, more, check, grid, eye, drag, settings, monitor.
Добавлять иконку — в этот файл, не инлайнить SVG в страницах, не использовать эмодзи.

## Базовые компоненты (UI-примитивы)

| Компонент | Назначение / props |
|-----------|--------------------|
| **Button** | variant: primary / secondary / ghost / danger; size: sm/md; `icon`, `loading`, `block`, `disabled` |
| **IconButton** | квадратная кнопка-иконка; variant; `aria-label` обязателен; `tooltip` |
| **Field** | обёртка label + хинт/ошибка вокруг Input/Select |
| **Input / Select** | fill-стиль (фон `--bg-sunken`, без рамки), фокус → белый фон + `--focus-ring` |
| **Card** | белая поверхность, `--radius-lg`, `--shadow-sm`, без рамки |
| **Panel** | боковая панель редактора/тепловых карт; заголовки секций `--font-label` |
| **Segmented** | переключатель режимов (вместо ряда обведённых кнопок): утопленный фон, активный — белая «таблетка» с тенью |
| **Modal** | кастомная (не нативный alert): оверлей + центр-карта, заголовок + close, действия внизу; деструктив подтверждается здесь |
| **Sheet** | нижняя шторка для **плеера** (тач): скруглённый верх, drag-handle, safe-area, тап-таргеты ≥44px — меню оператора |
| **Toolbar** | горизонтальная панель управления (тепловые карты): группы + segmented |
| **Toast** | ненавязчивое подтверждение («Сохранено»), автоскрытие |
| **EmptyState** | иконка + заголовок + подсказка + (опц.) действие — единый вид пустоты/загрузки |
| **Spinner** | индикатор загрузки |

Правила: **контекстное меню на тач (плеер) = Sheet**, не dropdown. Деструктив = Modal-подтверждение.
Новый компонент — когда паттерн повторился 3+ раз (анти-паттерн #4).

## Паттерны экранов
- **Главная**: холст, карточки прототипов (без рамок), «новый прототип» — кнопка → Modal с полями.
- **Редактор**: 3 колонки (панель экранов / стейдж / панель свойств) + тонкий топбар. Стейдж — `--bg-sunken`. Выбор/зоны — акцентом, не обводкой везде.
- **Плеер**: тёмный, устройство по центру, меню оператора — Sheet, иконки вместо эмодзи.
- **Тепловые карты**: панель (экраны/сессии) + Toolbar (Segmented: режим, фильтр) + стейдж с оверлеем. Цветовая легенда.

## Чек-лист ревью экрана
- [ ] нет хардкод-цветов/размеров — только токены
- [ ] нет эмодзи — только Icon
- [ ] минимум `border`: разделение фоном/тенью/воздухом
- [ ] loading / empty / error / success присутствуют
- [ ] действие даёт мгновенный отклик, нет двойного сабмита
- [ ] деструктив подтверждается (Modal), тач-меню — Sheet
